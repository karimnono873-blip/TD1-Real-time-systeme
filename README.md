# Correction TD Systèmes Temps Réel - Série 03

Ce dépôt contient la résolution détaillée et expliquée des exercices de la Série 03 pour le module **Systèmes Temps Réel**. La correction est fournie sous la forme d'une page web (HTML/CSS) stylisée pour une lecture claire et agréable.

## 📋 Contenu du TD

Le fichier HTML inclut les réponses, les calculs des facteurs d'utilisation (U) et l'analyse des diagrammes de Gantt pour les algorithmes d'ordonnancement suivants :

* **RM (Rate Monotonic)** et l'application du critère de Liu & Layland.
* **EDF (Earliest Deadline First)**.
* **LLF (Least Laxity First)**.
* **DMA (Deadline Monotonic Assignment)**.
* **Algorithmes classiques :** FCFS, Round Robin (RR), et SJF préemptif (SRTF).

## 🚀 Comment visualiser la correction

### Option 1 : En local
1. Clonez ou téléchargez ce dépôt sur votre ordinateur.
2. Double-cliquez sur le fichier `.html` pour l'ouvrir directement dans n'importe quel navigateur web (Chrome, Firefox, Safari, etc.).

### Option 2 : Via GitHub Pages
Si vous hébergez ceci sur votre propre compte GitHub, vous pouvez publier la page gratuitement :
1. Allez dans l'onglet **Settings** de votre dépôt GitHub.
2. Naviguez vers la section **Pages** (dans le menu de gauche).
3. Sous "Source", sélectionnez la branche `main` (ou `master`) et sauvegardez.
4. GitHub vous fournira un lien public pour consulter la correction en ligne.
